# React技术栈
## 原生开发中
- 要自己维护数据和DOM对象的一致。
- 不方便复用
  - 重复代码
  - 容易引起冲突
  - 代码比较零散，结构不清晰
  - 不方便重构

## 为什么要用React
- 组件化,把一个复杂的页面分成一个一个小组件
- 模块化
